# Holding Şirketi Web Sitesi

Candemir Holding web sitesine benzer, modern ve responsive bir kurumsal web sitesi.

## Özellikler

- ✅ Modern ve temiz tasarım
- ✅ Tam responsive (mobil uyumlu)
- ✅ Smooth scroll animasyonları
- ✅ Hero slider (otomatik geçiş)
- ✅ Dropdown menüler
- ✅ Mobil menü
- ✅ E-bülten kayıt formu
- ✅ Haberler/Blog bölümü
- ✅ İletişim formu

## Kullanım

1. `index.html` dosyasını bir web tarayıcısında açın
2. Veya bir local server kullanarak çalıştırın:
   ```bash
   # Python ile
   python -m http.server 8000
   
   # Node.js ile (http-server gerekli)
   npx http-server
   ```

## Dosya Yapısı

```
.
├── index.html      # Ana HTML dosyası
├── styles.css      # CSS stilleri
├── script.js       # JavaScript fonksiyonları
└── README.md       # Bu dosya
```

## Özelleştirme

- Renkleri değiştirmek için `styles.css` dosyasındaki `:root` değişkenlerini düzenleyin
- İçerikleri değiştirmek için `index.html` dosyasını düzenleyin
- Animasyonları ve interaktif özellikleri `script.js` dosyasından yönetin
